Notion to TOON - Options

Configure your Notion API integration to extract page content.

API Configuration

Your Internal Integration Token from Notion.

How to get your API key:

  1. Go to notion.so/my-integrations
  2. Click "New integration"
  3. Give it a name (e.g., "TOON Extractor")
  4. Select the workspace you want to access
  5. Click "Submit" to create the integration
  6. Copy the "Internal Integration Token" (starts with secret_)
  7. Paste it above and click Save

Important: You must share each page/database with your integration for it to have access. Open a Notion page, click the ••• menu, then "Add connections" and select your integration.
